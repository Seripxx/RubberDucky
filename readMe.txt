Not sure if you have to short the board before clicking USB > Open on Atmel. 
To short the board it is the pins next to the power/usb cable the top left if looking at the text on the board normally.
cover top left pin and then insert other side into ground hole.

click usb open.

choose hex file keyboard or usb.
click run.

If you want to change from keyboard to usb have to short the board.
